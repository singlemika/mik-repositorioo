🍴 KARIBOKA - Website Oficial
Este é um projeto de um site institucional e dinâmico para um restaurante, desenvolvido para oferecer uma experiência intuitiva aos usuários que desejam conhecer o cardápio, localizar o estabelecimento e entrar em contato.

🚀 Tecnologias Utilizadas
O projeto foi construído utilizando as tecnologias fundamentais da web (conhecidas como a "Trindade da Web"):

HTML :Estruturação semântica de todo o conteúdo.

CSS : Design responsivo, estilização visual e animações.

JavaScript

📋 Funcionalidades
Cardápio Interativo: Exibição organizada dos pratos e bebidas com fotos e descrições.

Design Responsivo: O site se adapta perfeitamente a dispositivos móveis, tablets e desktops.

Seção de Contatos: Integração de informações de telefone, e-mail e endereço.

Localização: Espaço dedicado para exibição do mapa ou endereço físico.

Navegação Suave: Âncoras inteligentes que facilitam a rolagem
